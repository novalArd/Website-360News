## ğŸš€ Cara Instalasi

1. **Clone repositori:**
   ```bash
   git clone https://github.com/novalArd/Website-360News.git
   ```

2. **Pindahkan folder ke direktori server lokal Anda**  
   Contoh untuk XAMPP:
   - Ekstrak atau salin folder ke dalam `htdocs` (Biasanya di : Local Disk(C) > xampp > htdocs)

